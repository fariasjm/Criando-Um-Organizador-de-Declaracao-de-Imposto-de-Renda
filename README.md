# Criando um Organizador de Declaração de Imposto de Renda

## Objetivo

Criação de uma planilha no Microsoft Excel para simplificar e organizar a coleta de informações para a elaboração da declaração anual de Imposto de Renda.

### Organização da Planilha
  - As barras de título e fórmula do Excel foram ocultadas para uma visualização mais limpa
  - Criados os botões do Menu ["Titular", "Informes", "Notas"]
  - Ocultados elementos de Interface (Barra de Títulos e Fórmulas)
  - Inserção e Formatação de Formas.

### Criação do Formulário "Titular"
  - O Menu "Titular" foi desenvolvida como um formulário para coletar dados pessoais
  - Campos como Nome, CPF, Nascimento, etc., foram inseridos e formatados
  - O botão "Próximo" foi criado, vinculado à próxima aba.

### Criação da Tela "Informes"
  - A aba "Informes" foi desenvolvida para registrar os informes bancários
  - Foram criados campos para "Banco", "Valor Atual" e "Anexo"
  - Os botões "Anterior" e "Próximo" foram adicionados.

### Criação da Tela "Notas"
  - Foi criado O último menu "Notas" para registrar entradas financeiras por categoria
  - Uma tabela foi configurada para registrar a "Data de Entrada", "Categoria" e "Valor"
  - O botão "Anterior" foi adicionado.
